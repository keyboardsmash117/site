Getting Started

You can be playing in about 30 minutes

1

Install Final Fantasy XI

You need an existing FFXI client installation with all expansions through Seekers of Adoulin. If you don't have one, install it through the official PlayOnline installer — you do not need an active subscription.

Once installed, download our DAT files and replace the ones in your FFXI installation folder. These contain updated client data needed to connect to Scythe.

Download DAT Files

Extract the downloaded files into your FFXI installation directory (typically C:\Program Files (x86)\PlayOnline\SquareEnix\FINAL FANTASY XI\), replacing existing files when prompted.

2

Download a Private Server Bootloader

Private servers use a bootloader to connect your FFXI client to our server instead of the official Square Enix servers. We recommend Ashita v4 — it's a free, open-source FFXI launcher with addon support, windowed mode, and quality-of-life features.

Windower is also supported if you prefer it. Both work fine. The key requirement is that the bootloader can redirect the FFXI client to a custom server address.

3

Create an Account

Account registration details are in our Discord server's #connection-info channel. The process takes about a minute. You'll create a username and password that you'll use to log in through the bootloader.

4

Configure the Connection

In your bootloader configuration (Ashita's scripts or Windower's settings), set the server address to connect to Scythe. The exact IP and port are provided in the #connection-info channel in Discord.

For Ashita v4, this means editing your boot script to point --server at our IP address. For Windower, you'll set the server field in the profile settings.

5

Launch and Play

Start your bootloader, log in with the account you created, and you're in Vana'diel. Create a character, pick a starting nation, and you're ready to go.

Check the #new-player-guide channel in Discord for tips on getting started — recommended first jobs, where to level, how to unlock trusts, and where to find your first hunt NMs.

Need Help?

If you get stuck at any step, ask in Discord. There's always someone around who can help with setup issues.
