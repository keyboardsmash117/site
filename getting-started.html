Getting Started

You can be playing in about 30 minutes

1

Install Final Fantasy XI

You need an existing FFXI client installation with all expansions through Seekers of Adoulin. If you don't have one, install it through the official PlayOnline installer — you do not need an active subscription.

Once installed, download our DAT files and replace the ones in your FFXI installation folder. These contain updated client data needed to connect to Scythe.

Download DAT Files

Extract the downloaded files into your FFXI installation directory (typically C:\Program Files (x86)\PlayOnline\SquareEnix\FINAL FANTASY XI\), replacing existing files when prompted.

2

Download a Private Server Bootloader

Private servers use a bootloader to connect your FFXI client to our server instead of the official Square Enix servers. We recommend Ashita v4 — it's a free, open-source FFXI launcher with addon support, windowed mode, and quality-of-life features.

Download Ashita v4

Windower is also supported if you prefer it. Both work fine. The key requirement is that the bootloader can redirect the FFXI client to a custom server address.

3

Create an Account

Account creation is self-service and takes about a minute. Because Scythe uses the standard LandSandBoat registration flow, you have two options:

  • Enter your desired username and password directly in the bootloader's login prompt on first connect — if the username doesn't exist yet, it is created automatically.
  • Or follow the current step-by-step pinned in Discord's #connection-info channel (occasionally updated — Discord is always authoritative).

You may run up to two accounts per player simultaneously. Keep your credentials safe — lost passwords cannot be recovered automatically.

4

Configure the Connection

In your bootloader configuration, set the server address to:

212.28.177.210

For Ashita v4, edit your boot script and set --server 212.28.177.210. For Windower, enter the IP in the server field in your profile settings.

5

Launch and Play

Start your bootloader, log in, and you're in Vana'diel. Create a character, pick a starting nation, and you're ready to go. Your character automatically receives every map and a server linkpearl on first login.

Your first move in-game: type !buff to turn on the levelling buff (scaling Regen/Refresh/Regain + 200% EXP), then !cc to warp to Chocobo Circuit — the custom hub zone with the auction house, crafting guild vendors, crystal moogle, Advanced Support NPC, and mission-item shop.

What's next?

Once you're logged in, the Guides page has a step-by-step "Your First Hour" walkthrough plus four recommended progression paths (new player, returning vet, solo, endgame hunter). If you get stuck at any step, ask in Discord — there's always someone around.
